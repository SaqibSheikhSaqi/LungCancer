# Early Lung Cancer Diagnosis Using Patient-Specific Indicators and Explainable AI

## Introduction
This study addresses the limitations of traditional Computer-Aided Detection (CAD) systems by focusing on non-imaging, patient-specific features such as:
- Age
- Smoking history
- Chest pain
- Existing medical conditions
- Anxiety levels

---

## About Dataset
The dataset used in this study is publicly available on Kaggle:

- **[Lung Cancer Dataset (Kaggle)](https://www.kaggle.com/datasets/ashwinchavan/lung-cancer-dataset)**

This dataset is designed for lung cancer prediction based on **patient-specific clinical and behavioral attributes**, rather than medical imaging. It contains multiple categorical features representing health conditions, lifestyle factors, and symptoms.

### Features Description

| Feature Name                | Description |
|----------------------------|-------------|
| **GENDER**                 | Gender of the patient (Male/Female) |
| **AGE**                    | Age of the patient |
| **SMOKING**                | Smoking habit status |
| **YELLOW_FINGERS**         | Indicator of nicotine staining (linked to smoking) |
| **ANXIETY**                | Presence of anxiety-related conditions |
| **PEER_PRESSURE**          | Influence of social environment on smoking behavior |
| **CHRONIC DISEASE**        | History of chronic illnesses |
| **FATIGUE**                | Level of fatigue experienced |
| **ALLERGY**                | Presence of allergies |
| **WHEEZING**               | Breathing difficulty or wheezing symptoms |
| **ALCOHOL CONSUMING**      | Alcohol consumption habits |
| **COUGHING**               | Persistent coughing condition |
| **SHORTNESS OF BREATH**    | Difficulty in breathing |
| **SWALLOWING DIFFICULTY**  | Trouble swallowing |
| **CHEST PAIN**             | Presence of chest pain |
| **LUNG_CANCER**            | Target variable (YES/NO) indicating lung cancer diagnosis |

### Dataset Characteristics
- The dataset consists of **categorical and numerical features**
- Primarily focuses on **symptoms and lifestyle factors**
- Suitable for **classification and risk prediction tasks**
- Enables **non-invasive, early-stage lung cancer detection modeling**

## Methodology

### Data Preprocessing
To improve model performance and reliability:
- **SMOTE (Synthetic Minority Over-sampling Technique)** was applied to balance the dataset.
- **Standard Scaling** was used to normalize feature values and ensure consistency.

### Model Implementation
A **Machine Learning Model** model was implemented due to its robustness and ability to handle complex feature interactions.

### Performance Metrics
The model achieved:
- **Accuracy:** 95%
- **Precision:** 95%
- **Recall:** 95%
- **F1-Score:** 95%

---

## Explainable AI Integration
To enhance interpretability and trust in predictions:
- **LIME (Local Interpretable Model-Agnostic Explanations)** was used for local explanations.
- **SHAP (SHapley Additive exPlanations)** was used for global feature importance analysis.

These tools allow patients and clinicians to:
- Understand diagnosis factors
- Analyze feature contributions
- Simulate lifestyle changes and their impact on risk level

---

## Counterfactual Analysis
To further enhance decision support, a **counterfactual analysis module** is incorporated to identify minimal changes in patient features that can alter the prediction outcome from **“Lung Cancer: YES” to “NO”** or vice versa.

### Objective
The goal is to answer:
- *What changes in a patient’s profile could reduce their predicted lung cancer risk?*
- *Which factors are most critical in flipping the model’s decision?*

### Method
Counterfactual instances are generated by slightly modifying key input features while keeping changes realistic and clinically meaningful. The model then evaluates whether these changes shift the prediction outcome.

### Example Scenarios
- **Smoking Reduction:** Changing smoking history from *heavy smoker* to *non-smoker* may shift prediction from YES → NO  
- **Anxiety & Health Management:** Improving anxiety levels and managing chronic conditions can reduce predicted risk  
- **Chest Pain Monitoring:** Reducing severity or frequency of chest pain may influence classification outcome  

### Interpretation
- Provides **actionable insights** rather than just predictions  
- Helps clinicians suggest **preventive lifestyle changes**  
- Enhances **patient understanding and engagement**  

### Benefit
Unlike static predictions, counterfactual analysis enables:
- Personalized risk reduction strategies  
- Transparent AI-driven recommendations  
- Simulation of “what-if” health scenarios
